# DraftBot

A Python bot for automating and managing **competitive drafts** — designed for use in Discord-based gaming communities.

## About

DraftBot streamlines the draft process for competitive games or fantasy leagues. It handles player pools, pick order, and team assignment, removing the manual overhead of running a draft in a Discord server.

## Tech

- **Language:** Python
- **Platform:** Discord (discord.py)

## Setup

1. Clone the repo.
2. Install dependencies: `pip install -r requirements.txt`
3. Add your bot token to `.env`.
4. Run: `python bot.py`
